# IPL Analytics Dashboard

## Project Overview
This interactive Power BI dashboard provides detailed insights into Indian Premier League (IPL) performance across seasons.
## Features

- Team Performance Analysis
- Player Performance
- Year-wise Statistics
- Interactive Filters
- KPI Cards
- Dynamic Visualizations

## Tools Used
- Power BI
- DAX
- Power Query
- Excel
## KPIs

- Total Runs
- Total Wickets
- Orange Cap
- Purple Cap
- Sixes
- Fours
- Matches
- Winning Percentage
